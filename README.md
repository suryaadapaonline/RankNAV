# RankNAV 🎓

RankNAV is a TG EAPCET (TS EAMCET) College Predictor that helps students estimate which engineering colleges and branches they may get based on their rank, category, gender, and preferences using previous years' cutoff data.

## 🌐 Live Demo

https://ranknav.vercel.app/

## ✨ Features

- Predict colleges using TG EAPCET rank
- Filter by category
- Gender-wise cutoff support
- Fast and responsive interface
- Previous years' cutoff based predictions
- Mobile-friendly design

## 🛠️ Tech Stack

- React
- TanStack Start
- TypeScript
- Tailwind CSS
- Vite
- GitHub
- Vercel

## 📊 Data Source

Predictions are based on publicly available previous years' TG EAPCET counselling cutoff data. Results are intended for guidance purposes only and should not be considered official.

## 🚀 Running Locally

```bash
git clone https://github.com/suryaadapaonline/RankNAV.git

cd RankNAV

npm install

npm run dev
```
